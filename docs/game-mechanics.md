# Game Mechanics

Parcel Society is a research game, not an entertainment game. Mechanics should remain simple, auditable, and reproducible.

## Season Length

Each experimental season lasts 7 rounds. Rounds create repeated opportunities for investment, cooperation, contracting, rent-seeking, and exit decisions.

## Action Points

Each participant receives 3 action points per round. Actions are intentionally limited so choices are interpretable and comparable across treatments.

## Parcel Map

Each server uses a 10x10 parcel map. Parcels will later receive quality levels according to the assigned inequality condition.

## Player Actions

The initial action vocabulary is:

- Produce
- Productive investment
- Safe asset
- Public contribution
- Informal contract
- Formal contract
- Lobbying
- Exit

These are placeholders for the research engine and should not be expanded without a study-design reason.

## Institutional Conditions

Stable institutions keep the rules and enforcement environment predictable across the season. Uncertain institutions introduce controlled uncertainty about institutional reliability while preserving deterministic seeded randomness for reproducibility.

## Inequality Conditions

Low inequality assigns relatively similar parcel opportunities across participants. High inequality assigns more unequal initial parcel opportunities. The exact allocation algorithms will be implemented later in the engine.
