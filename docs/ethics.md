# Ethics

Parcel Society should collect only the data required for research analysis.

- Use anonymous participant identifiers.
- Avoid unnecessary personal data.
- Do not include real-money payout logic in the MVP.
- Do not collect raw chat data; chat is outside MVP scope.
- Export only research-safe data needed for reproducible analysis.
- Document consent, debriefing, and institutional review requirements outside the application when used in real studies.
