# Admin Guide

The admin dashboard is not implemented in the foundation release. The intended workflow is:

1. Create a server for a study session.
2. Assign or randomize the experimental condition.
3. Generate the parcel map with deterministic seeded randomness.
4. Start the 7-round season.
5. Resolve each round after participant decisions are submitted.
6. Export research-safe CSV data for analysis.

Authentication, authorization, and operational audit logs will be designed before real study use.
